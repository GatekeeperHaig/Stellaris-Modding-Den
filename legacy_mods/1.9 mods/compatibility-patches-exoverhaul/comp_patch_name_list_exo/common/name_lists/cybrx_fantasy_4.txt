# Fricatives:	f v s z h
# Nasals:		m n ng nh
# Approximants:	l r
#
# Vowels: a e i
#
# <c> represents /k/
# <ch> represents pharyngealized /k/
# <h> represents a pharyngealized fricative
# <ng> represents a velar nasal

# Vowel system is a vertical vowel system
#
# Syllable Structure
#
# (C1)(l, r)V(C2)
#
# Where C2 is any of /d s n r l/
# Acceptable diphthongs: ie ia ea
#
