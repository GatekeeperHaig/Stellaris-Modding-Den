#############################################################################
#
# PHONOLOGY
#
# Stops: 		ph p b th t d ch c g
# Affricates:	x
# Fricatives:	v s h
# Nasals:		m n
# Approximants:	l r
#
# Vowels: a e i o u á é ó
# Diphthongs: ei éi ói ui au ei éi ói ui au áu éu eu iu óu ou uu ia ié ie iá ió iu ua ué ue uá ui uó
# Allowed initial consonant clusters: phr, pr, br, thr, tr, dr, chr, cr, gr, xr, phl, pl, bl, chl, cl, gl, xl
#
# <ph th ch> represent aspirated stops
# <x> represents a voiceless alveolar sibilant affricate
# <ch c> represent /k/
# <á é ó> represent an open front unrounded vowel, an open-mid front unrounded vowel, and an open-mid back rounded vowel respectively
# <a> represents an open back rounded vowel
#
# Syllable Structure
#
# CV(n, t, d, s, r, l)
#
