### Fantasy apostrophe_3
### This is kind of a parody of all those apostrophes in fantasy and sci'fi
### However, I quickly whipped up a conlang phonology for this one, so the apostrophes here actually make logical sense - in this case, the apostrophes represent ejective consonants
### For more on ejective consonants, and one way real world languages use apostrophes, see here: https://en.wikipedia.org/wiki/Ejective_consonant

#############################################################################
#
# PHONOLOGY
#
# Stops: 		p/p'/ph/t/t'/th/k/k'/kh
# Affricates:	c/c'
# Fricatives:	sh/s/s'/j/h
# Nasals:		m/n
# Approximants:	r/l/l'
#
# <l l'> represent Alveolar laterals
# <ph th kh> are aspirated stops
# <j> is a velar fricative (i.e. /x/)
# <c c'> represent /ts ts'/
# <sh> represents an aspirated /s/
#
# Syllable Structure
#
# (C1)V(C2)
#
# Where C2 is any of /t th s n r l/
#
# Uses postfixes in grammar
#
#############################################################################
